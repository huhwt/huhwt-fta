 * Extensions for webtrees admin actions
 * A) Check and display duplicate individuals in the database - MTV - MultTreeView
 * B) Check and display unrelated individuals in the database - UIC - Unconnected(IndividualsCheck)

This is a webtrees 2.2 module - It cannot be used with webtrees 2.1.

For webtrees 2.1 use the latest release v2.1.20.0 of the huhwt-mtv Branch 2.1.

Attention:
~~~
  This module requires to be operated in a PHP 8.3-onward system.
~~~

Description
-----------

In the Control Panel there is the range regarding the Family trees in your webtrees installation. In this range there is a slot offering functions for checking the Genealogy data. 2 of these functions have been enhanced and extended by displaying additional informations and actions as well as the option to transfer the XREFs of the displayed individuals to ['huhwt-cce'] for further analysis.

This module covers all the features of the former **'huhwt-mtv'** (which will no longer be maintained in the future) as described in 
the [description taken from there](/README-MTV.md).

This module introduces a redesigned view of "Find unrelated individuals".

##### Notes:
##### If the extension module ['huhwt-cce'](https://github.com/huhwt/huhwt-cce) is installed/activated, you can add the displayed people to the Clippings cart 'ℍ Clippings cart enhanced'.
